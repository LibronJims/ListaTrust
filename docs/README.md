# ListaTrust - Blockchain Credit Registry

An AI-Powered Blockchain Credit Registry System for Sari-Sari Stores in the Philippines.

## Features
- 🔐 Secure Authentication with OTP/MFA
- 📝 Immutable Blockchain Records
- 🤖 AI-Powered Trust Scoring
- 📊 Store Owner Dashboard
- 👑 Admin Panel with Audit Logs
- 🖼️ Profile Photo Upload

## Tech Stack
- **Backend:** Node.js, Express, MySQL
- **Frontend:** HTML5, CSS3, JavaScript
- **Security:** bcrypt, JWT, Rate Limiting, Input Validation
- **Blockchain:** Solidity Smart Contracts
